Atividade de PHP

    Vaquinha para festa junina

    By Giovanni Francesco Guarnieri

    Descrição da Atividade

    Desenvolva um sistema que simule uma vaquinha para uma festa junina. O sistema deve registrar:

    • Nome do colaborador
    • Valor da contribuição
    • Data da contribuição

    Código

    Requisitos básicos para o projeto ser considerado para avaliação:

    • O sistema deve ser feito utilizando: PHP, PDO e MySql.
    • Utilizar o banco de dados proposto abaixo.
    • Se utilizar imagens de terceiros não esquecer de citar os créditos.

    Critérios de avaliação

    • Estilização da página utilizando recursos de CSS(1,0)
    • Estruturação da página utilizando tags semânticamente adequadas para cada contexto(1,0)
    • Validação de dados utilizando isset no PHP e required no HTML (1,0)
    • Deve ser possível inserir (2,0)
    • Deve ser possível visualizar a lista de contribuições ordenado em ordem decrescente pela data de contribuição(2,0)
    • Deve ser possível remover contribuição (2,0)
    • Deve ser possível visualizar a quantidade total de arrecadações (0,5)
    • Deve ser possível visualizar o total em dinheiro arrecadado (0,5)

    Fique a vontade para adicionar mais funcionalidades ao sistema, porém, as funcionalidades acima são obrigatórias para avaliação.

    Embora o banco de dados seja o mesmo, dificilmente 2 grupos terão a mesma estruturação HTML, CSS e PHP. Deste modo, caso dois projetos tenham muita semelhança visual e estrutural, ambos terão a nota zerada.

    Banco de Dados

    Para a atividade, utilize o banco de dados abaixo: 

    
CREATE DATABASE vaquinha;
USE vaquinha;

CREATE TABLE colaborador (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL,
    valor DECIMAL(10,2) NOT NULL,
    data DATE NOT NULL
);
